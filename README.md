## 📖 About

**KEXER Information Toolkit** is a modern web application that provides multiple information lookup services through a clean, responsive, and interactive interface.

Built for **educational purposes**, **API integration practice**, and **web development learning**, the project demonstrates how multiple APIs can be integrated into a single professional dashboard.

---

## ✨ Features

- 📱 Phone Number Lookup
- 🆔 PAN Information
- 🚗 Vehicle RC Details
- 🏦 IFSC Code Verification
- 📧 Email Validation
- 📸 Instagram Information
- 🌍 IP Address Lookup
- 📍 Geolocation Tracking
- 🇵🇰 Pakistan Number Lookup
- 🎮 Free Fire UID Information
- 🚫 Free Fire Ban Status
- 🐙 GitHub Profile Lookup

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/karim2762/Tool-kit.git
```

Go to the project directory

```bash
cd Tool-kit
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
