# 📖 Overview

**KEXER Information Toolkit** is a modern information lookup platform that integrates multiple APIs into a single dashboard. It provides a clean Glassmorphism interface, responsive design, and fast API-based lookups, making it ideal for educational purposes and web development practice.

The project demonstrates:

- REST API Integration
- Python Backend Development
- Responsive Web Design
- Modern UI/UX Principles
- Real-Time Data Fetching
- Frontend & Backend Communication

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/karim2762/Tool-kit.git
```

Navigate to the project

```bash
cd Tool-kit
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
